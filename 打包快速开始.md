# 📦 Kronos 交易系统 - 打包快速开始

## 🚀 一键打包（3步完成！

### 第1步：确保环境准备

确保已安装所有依赖：
```bash
pip install -r requirements.txt
pip install pyinstaller
```

### 第2步：运行打包脚本

```bash
python build_exe.py
```

### 第3步：找到打包结果

打包完成后，在 `dist/` 目录下找到 `KronosTrading.exe`

## 🎯 使用提示

1. **首次运行**：exe 可能需要较长时间（解压模型）
2. **模型备份**：训练好的模型建议手动备份
3. **杀毒软件**：可能误报时添加信任
4. **保存位置**：建议将 `dist/KronosTrading.exe` 复制到其他位置运行
