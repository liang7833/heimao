# Qwen分析模块使用说明

## 功能概述

Qwen分析模块是黑猫交易系统的新增功能，使用**Qwen2.5-1.5B-Instruct**超轻量模型进行K线技术分析，与Kronos和FinGPT协同工作。

## 权重配置

默认权重比例：**Kronos 50% : Qwen 30% : FinGPT 20%**

## 使用方法

### 方法1：主面板开关（推荐）

1. 启动程序后，在左侧主面板的**"交易配置"**区域
2. 找到 **"🔮 启用Qwen分析"** 复选框
3. 勾选即可启用Qwen分析
4. 系统将自动：
   - 加载Qwen分析器
   - 同时向Kronos和Qwen提供币安数据
   - 按5:3:2权重融合三个模型的信号
   - 生成最终交易决策

### 方法2：AI策略中心开关

1. 进入**AI策略中心**标签页
2. 找到 **"🔮 Qwen分析模块配置"** 面板
3. 勾选 **"启用Qwen分析"** 即可

## 模型配置

### 下载Qwen2.5-1.5B-Instruct模型（推荐，超轻量）

运行以下命令自动下载：

```bash
py download_qwen_model.py
```

模型将自动下载到：
- Windows: `h:\kronos交易\models\qwen\`
- 或：`h:\kronos交易\_internal\models\qwen\`

### 使用Qwen3.5-4B（可选，需要最新依赖）

如果确实想用Qwen3.5-4B，请按以下步骤：

```bash
# 第一步：升级依赖
py -m pip install --upgrade transformers huggingface_hub
py -m pip install --upgrade torch

# 第二步：下载Qwen3.5-4B
py download_qwen_model.py --model Qwen/Qwen3.5-4B
```

### 规则分析模式（无需下载，推荐试用）

如果不想下载大模型，或者本地模型加载失败，系统将自动使用**基于规则的Qwen分析**作为备用方案，确保功能正常运行。

规则分析模式特点：
- ✅ 无需下载任何模型
- ✅ 立即可用
- ✅ 基于技术指标（MA、支撑阻力等）
- ✅ 与Kronos数据格式完全兼容

       │             │             │

## 开关状态

- **关闭**：恢复原有的Kronos+FinGPT分析模式
- **打开**：启用三模型协同分析（Kronos+Qwen+FinGPT）

## 常见问题

**Q: 模型加载失败怎么办？**
A: 请按顺序尝试：
1. 升级transformers: `py -m pip install --upgrade transformers huggingface_hub`
2. 确保有足够的显存
3. 或直接使用规则分析模式（无需模型）

**Q: 规则分析模式效果怎么样？**
A: 规则分析模式基于标准技术指标，对于普通交易已经足够。

**Q: 可以同时使用规则分析和模型吗？**
A: 程序会优先使用本地模型，如果没有则自动使用规则分析。

## 注意事项

1. 首次启用Qwen分析时，系统会初始化Qwen分析器，可能需要几秒钟
2. Qwen使用与Kronos完全相同的K线数据格式，无需额外转换
3. 开关状态实时生效，无需重启程序
4. 权重可在AI策略中心的"协调器参数"中调整
5. 完整Qwen2.5-1.5B-Instruct模型约3GB，建议4GB+显存
